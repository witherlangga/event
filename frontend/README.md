# Event Tickets - Flutter Frontend

This is a minimal Flutter frontend scaffold for the Event Tickets project.

Quick start (on your machine):

1. Ensure Flutter SDK is installed and on PATH.
2. From this repo root run:

```bash
cd frontend
flutter pub get
flutter run -d emulator-5554 # or your Android device id
```

The project is configured for Android only. Core packages are declared in `pubspec.yaml`.
